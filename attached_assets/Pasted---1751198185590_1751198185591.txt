Настройки заявок
Разрешить загрузку файлов
Позволяет пользователям прикреплять и загружать файлы к своим заявкам.
Разрешить гостевые заявки
Позволяет пользователям, у которых нет учетной записи, создавать заявки с созданием аккаунта и отправкой пароль на почту.
Разрешить редактирование заявки
Позволяет пользователям изменять свой собственную заявку (название, категорию, тело) после создания.
Требовать логин
Требует, чтобы пользователь вошел в свою учетную запись для доступа к клиентской области (гостевые заявки не разрешены).
Разрешить Рейтинг заявок
Позволяет пользователю оценить заявку в клиентской области.
Предотвращать ответы после закрытия заявки
Whenever you set the status of a ticket to close on View Ticket, a window will pop-up allowing you to enter ticket notes.

Настройки IMap
Эти строки должны находиться внутри шаблонов электронной почты, чтобы Cron мог получить необходимую информацию.

IMap Ticket String
## Номер заявки:
Эта строка используется для получения идентификатора заявки из сообщения электронной почты. В шаблоне это должно в начале строки перед ID
IMap Ticket Reply String
##- Введите свой ответ над этой строкой -##
Строка, используемая для определения конца ответа. Это должна быть полная линия.
